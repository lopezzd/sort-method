Projeto: Implementação e Análise de Algoritmos de Ordenação em C

Introdução

Caros alunos,

Neste projeto, vocês terão a oportunidade de mergulhar no fascinante mundo dos algoritmos de ordenação. A ordenação é uma operação fundamental na ciência da computação, e entender como diferentes algoritmos funcionam e se comportam é crucial para o desenvolvimento de software eficiente.

Objetivo do Projeto

O objetivo principal deste projeto é proporcionar a vocês uma experiência prática com diversos algoritmos de ordenação. Vocês irão:

  1. Implementar os algoritmos em C
  2. Medir e comparar seus desempenhos práticos
  3. Desenvolver habilidades de análise e apresentação de dados

Este projeto simulará um cenário do mundo real, onde vocês precisarão não apenas implementar soluções, mas também analisá-las e apresentar seus resultados de forma clara e profissional.

Tarefas Detalhadas

1. Implementação (30% da nota)

  • Implemente todos os seis algoritmos em C.
  • Use a mesma estrutura de dados (array de inteiros) para todos.
  • Certifique-se de que cada implementação seja correta e eficiente.
  • Comente seu código adequadamente para explicar a lógica de cada algoritmo.
Dica: Comece com os algoritmos mais simples (Bubble, Selection, Insertion) antes de passar para os mais complexos.

2. Testes de Performance (30% da nota)

  • Gere conjuntos de dados aleatórios com tamanhos variando de 1 a 100.000 elementos.
  • Teste cada algoritmo com os mesmos conjuntos de dados.
  • Use a biblioteca time.h para medir o tempo de execução (veja o exemplo de código no final).
  • Realize testes para pelo menos os seguintes tamanhos de array: 100, 1.000, 10.000, 50.000, 100.000.
Dica: Para resultados mais confiáveis, execute cada teste várias vezes e use a média dos tempos.

3. Análise Comparativa (15% da nota)

  • Compare o desempenho dos algoritmos para diferentes tamanhos de entrada.
  • Analise como o tempo de execução cresce em relação ao tamanho da entrada.
  • Discuta as diferenças entre o desempenho teórico (complexidade) e o prático observado.
  • Identifique em quais situações cada algoritmo se sai melhor.

4. Visualização de Dados (10% da nota)

  • Crie gráficos comparando o desempenho dos algoritmos.
  • Sugestão: use uma ferramenta como GNU Plot, Python (matplotlib) ou R.
  • Inclua pelo menos um gráfico de linha mostrando o tempo de execução vs. tamanho da entrada para todos os algoritmos.

5. Relatório (15% da nota)

Prepare um relatório em PDF contendo:

  • Introdução explicando o propósito do projeto
  • Breve descrição teórica de cada algoritmo
  • Metodologia: como você realizou os testes
  • Resultados: inclua suas tabelas e gráficos
  • Análise: interprete os resultados
  • Conclusão: o que você aprendeu, qual algoritmo é melhor em quais situações
  • Referências bibliográficas
  • Link para o repositório GitHub com o código-fonte

Entrega

  • Trabalho em grupos de até 3 pessoas.
  • Demonstração na aula prática da semana que vai de Entregue via Canvas até 10 de novembro às 23:59.
  • Envie o relatório em PDF e inclua o link do GitHub no relatório.

Algoritmos que foram implementados

  1. Selection Sort: Um algoritmo simples que repetidamente seleciona o menor elemento restante.
  2. Insertion Sort: Constrói a lista final um item de cada vez, como organizar cartas de baralho na mão.
  3. Bubble Sort: Compara pares adjacentes e os troca se estiverem fora de ordem.
  4. Merge Sort: Um algoritmo eficiente que divide a lista, ordena as partes e depois as mescla.
  5. Quick Sort: Um algoritmo rápido que usa a estratégia de dividir e conquistar.
  6. Heap Sort: Utiliza uma estrutura de dados chamada heap para ordenar os elementos.

Selection Sort

Para medir o tempo de execução, use o seguinte template:

#include <stdio.h>
#include <time.h>

void algumAlgoritmoDeOrdenacao(int arr[], int n) {
    // Implementação do algoritmo aqui
}

int main() {
    int arr[] = {64, 34, 25, 12, 22, 11, 90};
    int n = sizeof(arr) / sizeof(arr[0]);
    
    clock_t inicio, fim;
    double tempo_cpu;
    
    inicio = clock();
    algumAlgoritmoDeOrdenacao(arr, n);
    fim = clock();
    
    tempo_cpu = ((double) (fim - inicio)) / CLOCKS_PER_SEC;
    
    printf("Tempo de execução: %f segundos\n", tempo_cpu);
    
    return 0;
}

Insertion Sort

Para sortear números aleatórios, use o seguinte template:

#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {
    int arr[100000];
    int n = sizeof(arr) / sizeof(arr[0]);
    
    srand(time(NULL));

    for (int i=0; i<n; i++)
    {
        arr[i] = rand();
    }

    clock_t inicio, fim;
    double tempo_cpu;
    
    inicio = clock();
    algumAlgoritmoDeOrdenacao(arr, n);
    fim = clock();
    
    tempo_cpu = ((double) (fim - inicio)) / CLOCKS_PER_SEC;
    
    printf("Tempo de execução: %f segundos\n", tempo_cpu);
    
    return 0;
}
